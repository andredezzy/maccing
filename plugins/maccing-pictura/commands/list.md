# Pictura List

Use the `pictura_list` MCP tool to browse generated images.

## Flags

- `--limit`: Maximum batches to display (default: 10)
- `--filter`: Filter by prompt slug

## Examples

```
/pictura:list
/pictura:list --limit 5
/pictura:list --filter "product"
```
