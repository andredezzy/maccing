# Common Patterns

Ready-to-use box drawing patterns for common use cases.

---

## Simple Box (65 chars)

```
┌───────────────────────────────────────────────────────────────┐
│                           TITLE                               │
├───────────────────────────────────────────────────────────────┤
│                                                               │
│   Content line 1 with proper padding                          │
│   Content line 2 with proper padding                          │
│                                                               │
└───────────────────────────────────────────────────────────────┘
```

---

## Horizontal Flow

```
┌───────────┐      ┌───────────┐      ┌───────────┐
│  Step 1   │─────▶│  Step 2   │─────▶│  Step 3   │
└───────────┘      └───────────┘      └───────────┘
```

**Arrow**: `─────▶` (5× horizontal line + filled arrow)

---

## Vertical Flow

```
┌─────────────────┐
│     Parent      │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│     Child       │
└─────────────────┘
```

**Key**: Center `│` and `▼` under the box

---

## Branching Flow

```
     ┌─────────────┐
     │   Start     │
     └──────┬──────┘
            │
       ┌────┴────┐
       ▼         ▼
┌──────────┐ ┌──────────┐
│  Path A  │ │  Path B  │
└──────────┘ └──────────┘
```

## State Machine

```

          │

     ▼         ▼
┌────────┐ ┌────────┐

               │
```

## Decision Tree

```
                    ┌─────────────────┐
                    │   Is it null?   │
                    └────────┬────────┘
                             │
               ┌─────────────┴─────────────┐
               ▼                           ▼
        ┌─────────────┐             ┌─────────────┐
        │     YES     │             │     NO      │
        └──────┬──────┘             └──────┬──────┘
               │                           │
               ▼                           ▼
        ┌─────────────┐             ┌─────────────┐
        │ Return null │             │  Continue   │
        └─────────────┘             └─────────────┘
```

## Avoid These Patterns

| Pattern | Why |
|---------|-----|
| Width > 70 chars | Breaks in terminals, emails, GitHub |
| Heavy lines (━ ┃) | Inconsistent font support |
| Double lines (║ ═) | Complex, poor support |
| Mixed ASCII/Unicode | Different character widths |
| No breathing room | Looks cramped |
| Uncentered flow | Looks unprofessional |
